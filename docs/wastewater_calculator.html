폐수 분석 계산기

손글씨 계산 방식을 참고해 구성했습니다. 검량선 농도, 적정량, 시료량을 입력하면 결과와 계산식을 바로 확인할 수 있습니다.

1) 산/알칼리 혼합 pH

산과 알칼리 몰수 차이로 남는 성분을 계산한 뒤, 총 부피로 나누어 pH/pOH를 구합니다.
계산식: 남는 몰수 = |알칼리 - 산|, 농도 = 남는 몰수 / 총 부피
결과가 여기에 표시됩니다.

2) TN / TP 분석

검량선에서 구한 농도(mg)와 시료량을 입력합니다. 60/25는 실험법 고정 상수로 자동 반영됩니다.
고정값 60 / 25
농도 = 검량선 농도 × (60 / 25) × (1000 / 시료 사용량)
결과가 여기에 표시됩니다.

3) TP 전처리 없이

함수선 또는 검량선에서 구한 농도와 샘플 사용량만 입력합니다.
농도 = 검량선 농도 × (1000 / 샘플 사용량)
결과가 여기에 표시됩니다.

4) F⁻ / Phenol / CN

함수선 농도, 증류 후 총부피, 증류 시료량, 발색에 사용한 시료량을 입력합니다.
농도 = 검량선 농도 × (증류 후 총부피 / 증류 시료량) × (1000 / 발색 시료량)
결과가 여기에 표시됩니다.

5) NH3-N / Cr+6

검량선 농도와 시료량을 입력하면 mg/L로 환산됩니다.
농도 = 검량선 농도 × (1000 / 시료량)
결과가 여기에 표시됩니다.

6) Cl⁻

고정값: 0.1N AgNO3 1 mL = Cl 3.545 mg
고정값 3.545
Cl⁻ = (적정량 - 바탕시험량) × 3.545 × (1000 / 시료량)
결과가 여기에 표시됩니다.

7) N농도 적정

N1V1 = N2V2를 사용하며, 샘플의 N농도 X를 계산한 뒤 희석배수를 곱해 최종 N농도를 표시합니다.
N1 × V1 = X × V2 → X = (N1 × V1) / V2, 최종 N농도 = X × 희석배수
결과가 여기에 표시됩니다.

8) 폐기물 농도 계산

용출액 농도에서 폐기물 기준 mg/kg로 환산합니다.
폐기물 농도(mg/kg) = 용출액 농도(mg/L) × (1L / 1000mL) × 용출액량(mL) / 시료량(g) × (1000g / 1kg)
결과가 여기에 표시됩니다.

9) 총경도 및 Ca/Mg 환산

고정값: CaCO3 분자량 100g, Ca 환산계수 40/100, Mg 환산계수 24.3/100
EDTA 0.01M 1mL = CaCO3 1mg Ca = ×0.4 Mg = ×0.243
총경도 = (EDTA 적정량 - MgCl2 첨가량) × (1000 / 시료량)
결과가 여기에 표시됩니다.
계산 결과는 일반적으로 소수점 셋째 자리까지 표시합니다. pH 카드의 매우 작은 농도값은 유효숫자 3자리로 표시합니다.
